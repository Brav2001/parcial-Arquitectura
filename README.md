# Pacial Final Arquitectura

Crear la base de datos con el nombre

```
arquitectura
```

Copiar y pegar los scripts en la ruta

```
\back\scripts
```

## Backend

Ejecutar

```
cd back
npm i
npm run dev
```

## Frontend

Ejecutar

```
cd front
npm i @angular/cli
npm i
ng serve
```
